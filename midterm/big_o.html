Big-O

LinkedList, HashTable, Tree, Graph

LinkedList

A linear data structure that includes a series of connected nodes.

HashTable

A data structure that implements a dictionary (abstract data type that maps keys to values).

Tree

An abstract data type that represents a hierarchical tree structure with a set of connected nodes

Graph

A graph is a non-linear kind of data structure made up of vertices and edges.
