Sorting Algorithms

Insertion, Quick, Merge, Binary Search

Insertion

Considered inefficient on large lists compared to other sorting algos, insertion sort uses comparisons to sort arrays.

Quick

With the use of a selected pivot, quicksort partitions an array around the chosen pivot.

Merge

A recursive algorithm, mergesort uses a divide-and-conquer method to sort arrays.

Binary Search

This search algorithm requires an already sorted array to operate on.
